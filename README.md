# Spotify Dashboard

This is an interactive dashboard built using Plotly Dash to visualize the track scores from the "Most Streamed Spotify Songs 2024" dataset.

## Features

- View the distribution of track scores using:
  - Histogram
  - Box Plot
  - Violin Plot
